# aws configure
Use the command to set up your aws cli configurations on your PC
aws configure

Checking if we have s3 buckets - aws s3 ls
creating a bucket - aws s3 mb s3://bucket-name
Copying a file into a bucket: aws s3 cp file.txt s3://bucket-name
Deleting a file within an S3 bucket: aws s3 rm s3://bucket-name/file.txt
